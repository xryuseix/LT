# LT資料

[https://xryuseix.github.io/lt](https://xryuseix.github.io/lt)

* httpで特定スライドを起動

```sh
cd なんか/dist && php -S 127.0.0.1:8088
```

* httpで全てのスライドを起動

```sh
php -S 127.0.0.1:8088
```

* 開発環境で起動

```sh
cd なんか && yarn dev
```
